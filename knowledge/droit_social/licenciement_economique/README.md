## Usage

Cette base est indexée par le Retriever BM25 de Lucie. Elle est conçue pour :

1. La recherche lexicale rapide par mots-clés
2. L'alimentation des agents juridiques de Lucie V1
3. La couverture du périmètre licenciement économique pour le démo V1 (août 2026)
